# دليل مستخدم نظام قيد — الفهرس الرئيسي

> **قيد** — نظام المحاسبة الشخصية ذو السيادة المطلقة.
> محلي أولاً. مشفر بالكامل. دقيق رياضياً. مبني على عقيدة محاسبية صارمة.

---

## عن هذا الدليل

هذا الدليل ليس مجرد شرح للأزرار والشاشات. إنه شرح لمنطق نظام يفكر بطريقة مختلفة عن معظم تطبيقات المالية الشخصية. نظام قيد مبني على مبادئ راسخة، وكل قرار فيه — حتى القرارات التي قد تبدو تقييدية في البداية — موجود لسبب محاسبي أو أمني واضح.

قبل أن تبدأ في استخدام النظام، وقبل أن تتساءل "لماذا لا أستطيع فعل X"، اقرأ الفصول الأولى من هذا الدليل. ستجد إجاباتك هناك.
